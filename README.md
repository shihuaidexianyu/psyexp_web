# psyexp_web

`psyexp_web` 是一个面向心理学实验的多客户端网页实验平台。  
它采用单服务器、多客户端的架构，服务器负责实验流程控制、状态同步与数据记录，客户端通过浏览器接入并参与实验。

## 项目目标

本项目希望提供一个适合实验室环境使用的网页实验框架，支持：

- 单服务器、多客户端接入
- 不同客户端按角色参与实验
- 实验流程的统一调度与状态同步
- 行为数据与系统日志记录
- 局域网环境下的便捷部署与测试

## 适用场景

`psyexp_web` 适用于以下类型的任务：

- 多人行为实验
- 协作式或对抗式实验任务
- 基于浏览器的实验接入
- 实验室局域网环境下的多终端实验

## 当前设计

系统采用网页服务器模型：

- **Server**：负责会话管理、角色分配、实验状态机、消息广播、数据记录
- **Client**：通过浏览器访问，负责界面展示、输入采集、事件上报
- **Admin Panel**：用于实验创建、参数配置、客户端监控和数据导出

## 计划功能

- 房间 / 会话管理
- 角色分配与客户端接入
- WebSocket 实时通信
- 实验状态同步
- 键盘 / 鼠标行为采集
- 局域网自检与连接测试
- 行为数据导出
- 管理后台

## 技术栈

初步考虑使用：

- **Frontend**: React + TypeScript
- **Backend**: Python + FastAPI / Flask
- **Realtime Communication**: WebSocket
- **Storage**: SQLite / PostgreSQL
